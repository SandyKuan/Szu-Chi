Projects

SlowJimmy: Using a Text-Only Back-End to Power a Voice-Only Device
Tools: Python, HTML, CSS, Bootstrap, JavaScript, Flask, MySQL, Amazon Web Service, Alexa Skill
Crowd-AI Lab, Penn State University Dr. Kenneth Huang 2019 ~ 2021

This project introduces SlowJimmy, a prototype system that allows a human receptionist to converse with the user synchronously via an Amazon’s Echo device without access to the user’s voice recordings. SlowJimmy (i) populates possible alternative audio transcriptions to help the worker make sense out of noisy audio transcriptions and (ii) suggests follow-up lines automatically to help the worker produce responses quickly.

MealMatch: Crowdsourcing Restaurants from Food Photograph
Tools: Python, HTML, CSS, Bootstrap, JavaScript, Flask, MySQL, Amazon MTurk
Crowdsourcing & Crowd-AI Systems Dr. Kenneth Huang 2020

This project will build an end to end system which can allow users to upload food photographs and receive restaurants information within a few minutes. In addition, the restaurants information is generated from a series of tasks on Amazon Mechanical Turk.

Predict Future Sales (Kaggle Competition)
Tools: Python, Tensorflow, SKLearn, Pandas, Numpy, Seaborn, Matplotlib
Data Mining: Techniques and Applications Dr. Zhenhui (Jessie) Li 2019

This project aims to predict total sales for every product and store of a large Russian software firm using Machine Learning techniques. We worked with a challenging time-series dataset consisting of daily sales data and implemented several advanced algorithms as XGBoost, Long Short-Term Memory (LSTM), Random Forest and Convolutional neural network (CNN) on this dataset. Currently, our final prediction result is ranked top 15% on Kaggle leaderboard.
